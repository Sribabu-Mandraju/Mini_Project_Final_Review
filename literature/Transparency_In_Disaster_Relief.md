## 4. Increasing Donation Transparency in Disaster Relief: A Blockchain-based Solution

**Authors:** Mithilesh Lathkar, Parth Deshmukh, Aditya Patil, Dr. Priya Shelke
**Published:** ICETSIS 2024, IEEE — Vishwakarma Institute of Information Technology, Pune, India
<https://ieeexplore.ieee.org/document/10459402>

### Summary

This paper tackles the problem of opacity and potential corruption in disaster relief donation systems. Traditional centralized donation platforms lack real-time transparency, leaving donors unable to verify how their contributions are utilized.

The proposed system is a **decentralized donation tracking platform** built using **ReactJS**, **Web 3.0**, **Ethereum**, and **Solidity smart contracts**, designed to enable direct, transparent, and tamper-proof interactions between donors and relief campaigns.

**Key System Components:**

- **Actors:** Donors, Relief Agencies, Recipients, and Validators — each with distinct roles.
- **Smart Contracts (Solidity):** Four core functions were implemented — `createSupport()` for launching campaigns, `donateToSupport()` for processing donations directly to campaign owners, `getDonators()` for retrieving donor information, and `getSupports()` for listing all active campaigns.
- **MetaMask Integration:** Used for wallet management, secure authentication, and transaction confirmation.
- **UI:** Built with ReactJS and Tailwind CSS, offering a clean campaign browsing and donation experience.

**Benefits Demonstrated:** Every donation is verifiable and immutable on-chain, fraud risk is reduced, donor trust is improved, and the need for centralized intermediaries is eliminated.

**Conclusion:** The system provides a transparent, reliable, and accessible platform for disaster relief fundraising using blockchain's core properties of decentralization, traceability, and security.
